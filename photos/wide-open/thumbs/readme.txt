This folder is reserved for thumbnail images. Please include these if you wish to improve page loading.
You do not need to include thumbs but if you do wish to use them then please resize your image(s) and maintain
the original filename.

Recommended thumbnail sizes are:

640(width) x 480(height) for landscape
480(width) x 640(height) for portrait

For batch conversions please visit YouTube tutorial for Windows built-in batch resizing
